This repository contains the replication code for Rose, Schellenberg and Shem-Tov, Econometrica 2026.

Due to data privacy limitations, only simulated data is included. This simulated data will not reproduce the results in the paper. 

The Stata portions of the code were executed in Stata version 19 SE.. 

The python versions of the code were executed in version 3.14.3. We recommend you create a virtual environment and install required packages via the provided requirements file.
